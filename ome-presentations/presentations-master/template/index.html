Key Features

  • Each ‘panel’ is an independent viewer
  • Select multiple panels to co-edit
  • Full undo/redo
  • Save as orig file (json)
  • URL per file - bookmark, share, 'publish' etc
  • Script to create PDF from json
  • "Template" for figures

Tools & Tech

  • backbone.js (MVC - also angular.js, ember.js etc)
    • model
    • events
    • view
    • templating
    • router
  • Bootstrap - css, icons, js
  • Mousetrap - Keyboard bindings
  • Raphael - Drag handles etc
  • Reportlab - PDF generation
